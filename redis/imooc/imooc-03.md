## NoSQL的概述

`NoSQL`: not only NoSQL

为什么需要Nosql?

1. High performance - 高并发读写
2. High Storage - 海量数据的高效存储和访问
3. High Scalability && High Availability - 高可扩展性和高可用性

NoSQL数据库的四大分类

1. 键值（key-value）存储[redis]
2. 列存储［Hbase］
3. 文档数据库[mongoDB]
4. 图形数据库
